# 개발 체크리스트

## 1. 필수 기술 요소 ("이것이 자바다" 범위)

| 항목 | 적용 위치 | 확인 |
|------|----------|------|
| Collection (List, Map) | BusRouteList / SeatsMap / ReservationList | ☐ |
| File I/O (문자 스트림) | FileIOService.readCities / readSchedule | ☐ |
| 객체 직렬화 | FileIOService.readSeats / writeSeats / readReservation / writeReservation | ☐ |
| Stream API | SearchService.searchBusRoute / ReservationService.checkReservation | ☐ |
| 정규 표현식 | Main.inputUserId (`^[a-zA-Z0-9]{2,10}$`) | ☐ |
| 예외 처리 (try-catch-finally, throw) | FileIOService / ReservationService / Main 메뉴 루프 | ☐ |
| 열거형 (Enum) | Main 내부 MenuOption | ☐ |
| 람다식 | Stream filter 내부 | ☐ |
| 중첩 클래스 (선택) | Main 내부 enum / 필요 시 | ☐ |

## 2. 기능 동작 체크리스트

| 기능 | 확인 사항 | 결과 |
|------|----------|------|
| 프로그램 시작 | cities.txt / schedule.txt 로드 성공 | ☐ |
| 좌석 파일 없음 | seats.ser 없을 때 빈 좌석 초기화 | ☐ |
| 노선 검색 (출·도) | 정상 리스트 출력 | ☐ |
| 노선 검색 (출·도·시간) | 시간까지 포함 필터링 | ☐ |
| 좌석 조회 | 좌석 번호별 예약 여부 출력 | ☐ |
| 예약 생성 | 파일 저장 + 예약 번호 발급 | ☐ |
| 중복 좌석 예약 | 에러 메시지 + 재입력 | ☐ |
| 예약 조회 | 본인 예약만 출력 | ☐ |
| 예약 취소 | 좌석 해제 + 파일 갱신 | ☐ |
| 잘못된 ID 입력 | 정규식 검증 실패 → 재입력 | ☐ |
| 프로그램 재실행 | 이전 예약/좌석 복원 | ☐ |
